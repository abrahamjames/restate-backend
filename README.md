## Restate

> Restate is a real estate management platform that connect real estate agents and house seekers to connect

[Restate](https://github.com/nestjs/nest) is an estate management app.

## Technologies
- NestJs
- PostgreSQL
- JWT Passport

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
